# 기술 스택

## 플랫폼
- **모바일 최적화 웹** (반응형, 모바일 퍼스트)
- 별도 앱 없이 웹 브라우저로 접근
- 모든 주요 브라우저 지원 (Chrome, Safari, Firefox, Edge — 모바일 포함)

---

## 녹음 → 텍스트 변환: Gemini 2.0 Flash (무료)

### 추천 모델
**Gemini 2.0 Flash** — 무료 tier에서 오디오 입력을 직접 지원하는 멀티모달 모델

### 지원 오디오 포맷
- MP3, WAV, AAC, OGG, FLAC, AIFF, WebM

### 무료 tier 제한
| 항목 | 제한 |
|------|------|
| 분당 요청 수 | 15건 |
| 일일 요청 수 | 1,500건 |
| 파일 크기 (inline) | 20MB |
| 파일 크기 (Files API) | 2GB |
| 오디오 길이 | 최대 약 9.5시간 |

> 소규모 교회 선교 활동 기준 무료 tier로 충분.

### 한국어 지원
- 한국어 음성 인식 및 전사 **가능**
- 프롬프트에 "한국어로 전사" 명시 권장

### 녹음 → 텍스트 변환 파이프라인

```
[녹음 시작] 버튼 (브라우저 MediaRecorder API)
    ↓
마이크 권한 허용 → 실시간 녹음
    ↓
[녹음 중지] → 녹음 파일 생성 (webm/mp4)
    ↓
Supabase Storage에 업로드 (저장)
    ↓
녹음 파일 + 프롬프트 → Gemini 2.0 Flash API 전송
    ↓
Gemini가 일지 양식에 맞춰 JSON으로 분류하여 반환:
{
  "날짜": "2026-04-02",
  "만남장소": "학교 카페",
  "생명반응": "복음에 대해 긍정적 반응. 다음 주 다시 만나기로 함."
}
    ↓
JSON → 일지 작성 폼에 자동 입력
    ↓
사용자 확인/수정 후 저장
```

### 녹음 파일 업로드 방식도 동일
```
기존 녹음 파일 선택 (mp3/wav 등)
    ↓
Supabase Storage에 업로드
    ↓
이하 동일 (Gemini API → JSON → 폼 자동 입력)
```

### 주의사항
- Gemini는 전용 STT가 아닌 LLM 기반이라 가끔 요약/의역할 수 있음
- 프롬프트에 "정확히 전사" 강조 필요

### Rate Limit 대응: 작업 큐 시스템

무료 tier는 분당 15건 제한이 있으므로, 여러 요청이 동시에 들어와도 안정적으로 처리하기 위해 **백엔드 작업 큐**를 사용한다.

#### 동작 흐름
```
사용자가 녹음 제출
    ↓
DB에 작업 등록 (상태: 대기중)
    ↓
사용자에게 "변환 중입니다" 표시
    ↓
백엔드 큐 워커가 대기열에서 하나씩 꺼내서 처리
    ↓
Gemini API 호출 (rate limit 간격 유지: 최소 4초 간격)
    ↓
변환 완료 → DB에 결과 저장 (상태: 완료)
    ↓
사용자에게 알림 또는 페이지 새로고침 시 결과 표시
```

#### 작업 큐 테이블 구조 (개념)
| 필드 | 설명 |
|------|------|
| id | 작업 ID |
| 녹음파일_url | Supabase Storage 경로 |
| 생명_id | 연결된 생명 |
| 요청자_id | 요청한 사용자 |
| 상태 | 대기중 / 처리중 / 완료 / 실패 |
| 결과_json | Gemini 변환 결과 |
| 생성일시 | 요청 시각 |
| 완료일시 | 처리 완료 시각 |

#### 핵심 규칙
- **분당 최대 15건** 이내로 처리 (4초 간격)
- 일일 1,500건 초과 시 다음 날까지 대기
- 실패 시 자동 재시도 (최대 3회)
- 사용자는 대기 상태에서도 직접 작성으로 전환 가능

## 저장소 / DB: Supabase

- **Database**: 생명, 사용자, 일지, 강의 체크 등 전체 데이터
- **Storage**: 녹음 파일 저장
- **Auth**: 회원가입/로그인 (아이디 + 4자리 비밀번호)

---

## 프론트엔드: Next.js

- **Next.js** (React 기반, App Router)
- **Tailwind CSS** (모바일 퍼스트 반응형 스타일링)
- Supabase JS 클라이언트로 DB 연동
- Gemini API는 Next.js API Route (서버사이드)에서 호출 — 키 노출 방지
