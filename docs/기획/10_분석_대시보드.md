# 분석 대시보드

## 접근 권한
- **관리자**: 자기 소속 대학생/생명 범위 분석
- **강사**: 전체 데이터 분석

---

## 분석 항목

### 1. 단계별 현황 (퍼널 분석)
전체 생명이 어느 단계에 몇 명 있는지 한눈에 파악.

```
1차 만남: ████████████ 35명
전초:     ████████     22명
입문:     ██████       18명
초급:     ████         12명
중급:     ███           8명
고급:     ██            5명
수료:     █             2명
페일:     ██████       16명
```

> **어디서 막히는지** — 특정 단계에서 정체된 생명이 많으면 병목 구간 파악 가능

### 2. 병목 분석
- 각 단계에 **오래 머물러 있는 생명** 목록 (마지막 업데이트 기준)
- 예: "전초 단계에서 30일 이상 진행 없는 생명 5명"
- 클릭 시 해당 생명 상세 페이지로 이동

### 3. 강의 진행 분석
- 강사 연결된 생명 중 **강의를 잘 듣고 있는지** 추적
- 강의별 수강 완료율 (30개 강의 각각 몇 %가 들었는지)
- 특정 강의에서 많이 멈추는지 파악
- 패스 비율 (어떤 강의가 자주 패스되는지)

### 4. 대학생별 성과
- 대학생별 등록 생명 수
- 대학생별 활성/페일 비율
- 대학생별 평균 진행 단계

### 5. 관리자별 현황
- 관리자별 소속 대학생 수, 생명 수
- 관리자별 단계 분포

### 6. 기간별 추이
- 월별/주별 신규 생명 등록 수
- 월별/주별 단계 전환 수
- 월별/주별 페일 수

## UI 구성

관리자/강사 조직도 페이지에서 **[분석]** 탭 또는 버튼으로 진입.

### 화면 구성 예시
```
┌──────────────────────────────────────────────────┐
│  [조직도]  [분석]                                 │
├──────────────────────────────────────────────────┤
│                                                  │
│  ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐    │
│  │ 전체   │ │ 전초   │ │ 강의중 │ │ 수료   │    │
│  │  118명 │ │  22명  │ │  43명  │ │   2명  │    │
│  └────────┘ └────────┘ └────────┘ └────────┘    │
│  ┌────────┐                                      │
│  │ 페일   │                                      │
│  │  16명  │                                      │
│  └────────┘                                      │
│                                                  │
│  단계별 현황 (막대 그래프)                        │
│                                                  │
│  병목 알림                                       │
│  "전초 30일+ 정체: 5명"                          │
│                                                  │
│  강의 진행률                                     │
│                                                  │
│  대학생별 성과 요약                              │
│                                                  │
└──────────────────────────────────────────────────┘
```

### 상단 요약 카드
| 카드 | 설명 |
|------|------|
| **전체** | 연결된 전체 생명 수 (페일 제외) |
| **전초** | 관리자가 전초 진행 중인 생명 수 |
| **강의중** | 강사 연결 후 강의 수강 중인 생명 수 (입문~고급) |
| **수료** | 전체 강의 과정 수료 완료한 생명 수 |
| **페일** | 포기/실패 처리된 생명 수 |
