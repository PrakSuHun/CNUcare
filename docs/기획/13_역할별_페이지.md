# 역할별 페이지 구성

## 지도자 (/leader)
| 탭 | 기능 |
|----|------|
| 조직도 | 전체 조직도 + 편집(이동/삭제) + 검색/필터 |
| 내 생명 | 직접 관리 생명 목록 + 추가/해제/삭제 |
| 캘린더 | 일정 관리 + 생명 약속 연동 |
| 현황 | 대시보드 (요약카드, 퍼널, 병목, 성과) |
| AI 분석 | 4종 보고서 + AI 채팅 |

## 관리자 (/manager)
| 탭 | 기능 |
|----|------|
| 조직도 | 내 소속/전체 토글 + 편집 + 알림(노란점) + 전체읽음 |
| 내 생명 | 직접 관리 생명 목록 |
| 캘린더 | 일정 관리 |
| 현황 | 대시보드 |
| AI 분석 | 4종 보고서 + AI 채팅 |

### 관리자 알림
- 소속 대학생이 일지 작성 → 해당 생명에 노란 점 표시
- 생명 상세 진입 시 자동 읽음 처리
- 전체 읽음 버튼

## 강사 (/instructor)
| 탭 | 기능 |
|----|------|
| 내 강의 생명 | 연결된 생명 목록 (읽기 전용 상세) |
| 캘린더 | 강의 일정 관리 |
| AI 분석 | 생명 분석 + 강의 반응 정리 |

### 강사 특징
- 생명 자동 연결: 일지에 강의자 이름 입력 시
- 수동 연결: 생명 검색 → 연결
- 생명 상세: 읽기 전용 (수정/삭제 불가, 열람만)
- 일지 상세: 읽기 전용 (JournalView)
- 연결 해제만 가능 (영구 삭제 불가)

## 대학생 (/student)
| 탭 | 기능 |
|----|------|
| 내 생명 | 생명 목록 + 추가/연결해제/삭제 |
| 캘린더 | 일정 관리 |

## 어드민 (/admin)
| 탭 | 기능 |
|----|------|
| 조직도 | 전체 조직도 + 편집 |
| 캘린더 | 일정 관리 |
| 현황 | 대시보드 |
| AI 분석 | 4종 보고서 + AI 채팅 |
| 권한 관리 | 사용자 역할 변경 |
