# CNUcare 서비스 개요

## 프로젝트 목적
교회 선교 활동에서 **생명**(선교 대상자)의 등록부터 전초, 말씀 교육까지의 전 과정을 관리하고, 다수의 생명 진행 상황을 한눈에 파악할 수 있는 웹 서비스.

---

## 사용자 역할 (5종)

| 역할 | 설명 |
|------|------|
| **지도자** | 최고 권한. 조직도(편집), 내 생명, 캘린더, 현황, AI 분석 |
| **관리자** | 팀 관리. 조직도(편집, 알림), 내 생명, 캘린더, 현황, AI 분석 |
| **강사** | 강의 전문. 내 강의 생명(읽기전용), 캘린더, AI 분석(생명분석+강의반응) |
| **대학생** | 전도 실행. 내 생명 목록, 캘린더 |
| **어드민** | 시스템 관리. 조직도(편집), 캘린더, 현황, AI 분석, 권한 관리 |

### 역할 서열
지도자 > 관리자 > 강사 > 대학생

---

## 핵심 엔티티

### 생명 (Life)
- 선교 대상자 한 명
- 대학생이 최초 등록 (이름만으로 생성 가능)
- 여러 사용자가 연결 가능 (N:N)

### 일지 (Journal)
- 생명에 대한 기록 (날짜, 장소, 생명반응)
- 목적: 1차 만남 / 전초 / 관리 / 강의
- 강의 시 강의자 이름 + 수강 강의 선택 → 진도표 자동 연동
- 녹음 → Supabase 저장 → Gemini 텍스트 변환

### 약속 (Appointment)
- 생명과의 만남 일정
- 목적: 전초 / 관리 / 강의
- 전초 시 관리자 연결 / 직접 / 다른 사람 연결 선택
- 캘린더와 양방향 연동

### 강의 진도표 (Lesson Check)
- 30강 + 부가 과정 체크
- 일지에서 강의 작성 시 자동 체크 + 상세(강사, 반응) 연동
- 체크/해제 직접 가능

---

## 단계 자동화

| 트리거 | 단계 변경 |
|--------|----------|
| 생명 등록 | → 1차 만남 |
| 약속에서 전초 약속 생성 | → 전초연결 |
| 일지에서 전초 작성 | → 전초 |
| 일지에서 강의 작성 | → 입문 |
| 진도표 입문 5강 완료 | → 초급 |
| 진도표 초급 5강 완료 | → 중급 |
| 진도표 중급 9강 완료 | → 고급 |
| 수료 버튼 (수동) | → 수료 (수료일 입력) |

단계는 앞으로만 자동 변경. 진도표 체크 해제 시 뒤로도 조정.

---

## 핵심 가치
> **"한눈에 보이는 진행 상황"**
> 각 생명의 현재 단계와 최근 일지를 대시보드/조직도에서 한눈에 파악.

## 기술 스택
- **프론트엔드**: Next.js 16 + TypeScript + Tailwind CSS
- **DB/Storage**: Supabase (PostgreSQL + Storage)
- **AI**: Gemini 2.5 Flash/Pro (녹음변환, 채팅) + Claude Sonnet (분석 보고서, 충남대 Gateway)
- **배포**: Vercel
- **모바일 최적화**: 반응형 웹 (모바일 퍼스트)
