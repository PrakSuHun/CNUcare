# 강의 커리큘럼 및 진도 체크

## 강의 진도표 구조

생명별로 강의 진도표가 존재하며, 강사가 강의 체크를 기록한다.

### 진도표 헤더 정보
| 항목 | 설명 |
|------|------|
| 이름 | 생명 이름 |
| 전도자 | 담당 대학생 |
| 소속 | 소속 교회/기관 |

### 강의 체크 기록 필드
| 필드 | 설명 |
|------|------|
| 수강일 | 해당 강의를 들은 날짜 |
| 강의자 | 강의를 진행한 강사 이름 |
| 비고 | 특이사항 메모 (예: "아직 전조만(선택과 분별)") |
| 패스 | 강의를 듣지 않고 넘어가는 경우 체크. **패스 시 수강 완료로 집계됨** |

## 레벨 자동 반영 규칙

생명 목록의 단계 표시는 강의 체크 기록에 따라 자동 반영.
**해당 등급의 강의를 모두 완료해야 다음 등급으로 전환된다.**

| 단계 | 전환 조건 |
|------|----------|
| 강사 연결 (입문) | 강사 연결 후 초기 상태. 1~5번 수강 중 |
| 초급 | 입문 5강 **모두 완료** + 기도 교육 완료 시 전환 |
| 중급 | 초급 5강 **모두 완료** 시 전환 |
| 고급 | 중급 9강 **모두 완료** + 예배 교육 완료 시 전환 |
| 수료 | 고급 11강 **모두 완료** + 악평 교육 완료 시 전환 |

### 부가 과정도 시스템에서 관리
- **기도 교육**: 입문과 초급 사이 — 체크 항목으로 관리
- **예배 교육**: 중급과 고급 사이 — 체크 항목으로 관리
- **악평 교육(교리 비교)**: 고급 이후 — 체크 항목으로 관리
- **역사론**: 최종 과정 — 체크 항목으로 관리
- **예배 참석일**: 주일예배/수요예배 (대면/비대면) — 날짜 기록
- **성경 읽기**: 읽기 현황 기록
