# 단계 자동화

## 단계 정의
```
1차 만남 → 전초연결 → 전초 → 입문 → 초급 → 중급 → 고급 → 수료
```

## 자동 변경 규칙

| 트리거 | 새 단계 | 조건 |
|--------|---------|------|
| 생명 등록 | 1차 만남 | 기본값 |
| 약속에서 전초 생성 | 전초연결 | 현재 < 전초연결 |
| 일지에서 전초 작성 | 전초 | 현재 < 전초 |
| 일지에서 강의 작성 | 입문 | 현재 < 입문 |
| 입문 1~5강 모두 완료 | 초급 | 진도표 기반 |
| 초급 6~10강 모두 완료 | 중급 | 진도표 기반 |
| 중급 11~19강 모두 완료 | 고급 | 진도표 기반 |
| 수료 버튼 클릭 | 수료 | 수동만, 수료일 입력 |

## 핵심 규칙
1. **앞으로만 자동 변경** (약속/일지 트리거)
2. **진도표는 양방향** — 체크 해제 시 단계가 뒤로 조정됨
3. **수료는 수동만** — 수료 버튼 + 수료일 입력
4. **수료 후 변경 불가** — 수료 상태는 자동 변경 대상 아님
