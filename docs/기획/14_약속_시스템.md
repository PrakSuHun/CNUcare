# 약속 시스템

## 개요
생명 상세 페이지의 [약속] 탭에서 만남 일정을 관리하고, 캘린더와 양방향 연동.

---

## 약속 목적 (3종)

### 전초
- 3가지 선택:
  - `[관리자 이름] 연결` — 소속 관리자에게 자동 연결, 캘린더 공유
  - `직접 전초` — 본인이 직접
  - `다른 사람 연결` — 이름 입력, 매칭 시 캘린더 공유
- 약속 생성 시 생명 단계 → **전초연결** 자동 변경

### 관리
- 전초와 동일한 담당자 선택 구조

### 강의
- 강의자 이름 입력
- 이름이 강사 계정과 매칭 시 자동 연결 + 캘린더 공유

---

## 약속 필드
| 필드 | 설명 |
|------|------|
| 목적 | 전초 / 관리 / 강의 |
| 날짜 | 필수 |
| 시간 | 선택 |
| 장소 | 선택 |
| 담당자/강의자 | 목적에 따라 |
| 메모 | 선택 |

## UI
- 예정 약속: 클릭 시 수정 가능
- 지난 약속: 접힘 (details)
- 삭제 버튼
- readOnly 모드 (강사)

## 양방향 연동
- 생명 약속 탭에서 추가 → 캘린더에 표시
- 캘린더에서 추가 → 생명 약속 탭에 표시
