# 캘린더 시스템

## 개요
모든 역할(대학생/관리자/강사/지도자/어드민)이 사용하는 일정 관리 도구.
생명 약속과 양방향 연동되며, 개인일정/행사 기능 포함.

---

## 일정 유형 (5종)

### 1. 전초 / 관리 / 강의 (생명 연결)
- 생명 약속 탭과 동일한 구조
- 생명 선택 필수
- 양방향 연동 (캘린더 ↔ 생명 약속 탭)

### 2. 개인일정
- 생명 선택 불필요
- 본인만 볼 수 있음
- 필드: 제목, 날짜, 시간, 장소, 메모

### 3. 행사
- 그룹 일정 (여러 사람과 공유)
- 필드: 제목(예: 노방, 동아리, 큐티), 공유 멤버, 날짜, 시간, 장소, 메모
- 행사 생성 시: **새로 만들기** / **기존 행사 선택**
- 멤버 추가: 이름 검색 → 추가 (여러 명 가능)
- 공유된 멤버의 캘린더에 자동 표시

---

## 그룹 달력

### 구조
- 행사를 만들면 **그룹 달력**이 생성됨
- 그룹 달력 이름 = 행사 제목

### 관리 UI (캘린더 하단 또는 사이드)
```
그룹
☑ 노방        [해제]
☑ 동아리      [해제]
☐ 큐티        [연결]
+ 그룹 검색/연결
```

- **체크박스**: 캘린더에 표시할지 토글
- **해제**: 그룹에서 나가기
- **연결**: 검색해서 기존 그룹에 참여
- **검색**: 그룹 이름으로 검색

## DB 설계

### personal_events (개인일정)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| user_id | uuid | 작성자 |
| title | text | 제목 |
| date | date | 날짜 |
| time | text | 시간 |
| location | text | 장소 |
| memo | text | 메모 |
| created_at | timestamptz | 생성일 |

### event_groups (행사/그룹)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| name | text | 행사 이름 (그룹 달력 이름) |
| created_by | uuid | 생성자 |
| created_at | timestamptz | 생성일 |

### event_group_members (그룹 멤버)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| group_id | uuid | FK→event_groups |
| user_id | uuid | FK→users |
| is_visible | boolean | 캘린더에 표시 여부 (기본 true) |
| joined_at | timestamptz | 참여일 |

### group_events (행사 일정)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| group_id | uuid | FK→event_groups |
| title | text | 일정 제목 |
| date | date | 날짜 |
| time | text | 시간 |
| location | text | 장소 |
| memo | text | 메모 |
| created_by | uuid | 작성자 |
| created_at | timestamptz | 생성일 |

## 캘린더 UI

### 월간 달력
- 날짜에 일정 있으면 색상 점 표시
  - 파란색: 생명 약속
  - 초록색: 개인일정
  - 주황색: 행사
- 날짜 클릭 → 해당 일 일정 목록

### 일정 추가
```
[전초] [관리] [강의] [개인] [행사]
       ↓ 선택 후
    해당 유형 폼 표시
```

### 하단: 그룹 관리
```
그룹
☑ 노방
☑ 동아리  
[+ 그룹 검색/연결]
```
