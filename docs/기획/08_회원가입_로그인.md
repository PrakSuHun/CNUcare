# 회원가입 및 로그인

## 회원가입

### 가입 흐름
```
로그인 페이지 → [가입하기] 버튼 클릭 → 회원가입 페이지
```

### 입력 항목
| 필드 | 필수 | 설명 |
|------|------|------|
| 아이디 | 필수 | 사용자가 직접 설정. 중복 불가 |
| 비밀번호 | 필수 | 4자리 숫자 |
| 이름 | 필수 | 실명 |
| 생년월일 | 필수 | 생년월일 입력 |
| 전화번호 | 필수 | 연락처 |

### 역할 설정
- 가입 시 기본 역할은 **대학생**
- 관리자/강사 역할은 **어드민이 부여**

---

## 로그인

### 로그인 페이지 구성
- **아이디** 입력란
- **비밀번호** (4자리) 입력란
- **[로그인]** 버튼
- **[가입하기]** 버튼 → 회원가입 페이지로 이동

---

## 표시 이름 규칙

동명이인 구분을 위해 시스템 내 표시 이름은 **이름 + 생년 뒷 2자리** 형태:

```
박수훈00    → 2000년생 박수훈
김민지02    → 2002년생 김민지
이영희99    → 1999년생 이영희
```

- 생명 목록, 조직도, 일지 작성자 등 **시스템 전체에서 이 형식으로 표시**
- 생명(선교 대상자)의 표시 이름도 동일 규칙 적용

---

## 어드민 (최고 관리자)

별도의 **어드민 페이지**에서 전체 사용자의 역할을 관리한다.

### 어드민 기능
- 전체 가입된 사용자 목록 조회
- 사용자의 역할 변경: 대학생 → 관리자 / 대학생 → 강사 / 역할 회수
- 어드민 계정은 시스템에 미리 지정 (사이트 운영자)

### 역할 변경 흐름
```
어드민 페이지 로그인
    ↓
사용자 목록에서 대상 선택
    ↓
역할 변경 (대학생 / 관리자 / 강사)
    ↓
저장 → 해당 사용자 다음 로그인부터 변경된 역할로 접근
```

## 미결 사항
- [ ] 비밀번호 변경/찾기 기능 필요 여부
- [ ] 아이디 형식 제한이 있는지 (영문만, 길이 제한 등)
- [ ] 어드민 계정 로그인 방식 (별도 URL? 같은 로그인에서 어드민 화면 진입?)
