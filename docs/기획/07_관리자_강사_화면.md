# 관리자 / 강사 화면 흐름

## 역할별 권한 비교

| 항목 | 대학생 | 관리자 | 강사 |
|------|--------|--------|------|
| 볼 수 있는 생명 | 자신에게 연결된 생명만 | **모든 생명** | **모든 생명** |
| 목록 형태 | 단순 리스트 | 조직도(트리) | 조직도(트리) |
| 조직 편집 | X | **자기 밑 대학생 배치 + 생명 재배치** | **전체 모든 권한** |
| 검색/필터 | X | X | **O** |

---

## 1. 로그인 페이지
- 대학생과 동일하게 이름 입력으로 로그인
- 역할(관리자/강사)에 따라 다른 화면으로 진입

---

## 2. 생명 조직도 페이지 (메인)

관리자와 강사 모두 **동일한 조직도 화면**을 본다.
모든 생명을 **관리자 → 대학생 → 생명** 트리 구조로 표시.

### 조직도 구조
```
관리자 A
├── 대학생 1
│   ├── 김OO | 20세 | 컴퓨터공학과 | 전초   | 03.28
│   ├── 이OO | 22세 | 경영학과    | 초급   | 03.25
│   └── 박OO | 21세 | 국문학과    | 페일   | 03.10
├── 대학생 2
│   ├── 최OO | 19세 | 물리학과    | 입문   | 04.01
│   └── 정OO | 23세 | 화학과      | 중급   | 03.30
│
관리자 B
├── 대학생 3
│   └── ...
```

### 생명 항목에 표시되는 정보
| 항목 | 설명 |
|------|------|
| 생명 이름 | 클릭 시 생명 상세 페이지로 이동 |
| 나이 | 생명의 나이 |
| 학과 | 소속 학과 |
| 상태 | 현재 단계 (1차 만남 / 전초 / 입문 / 초급 / 중급 / 고급 / 수료 / 페일) |
| 마지막 만난 날짜 | 날짜만 표시 (설명 없이) |

### 폴더 형태 동작
- **관리자 이름**: 펼치면 소속 대학생 목록 표시
- **대학생 이름**: 펼치면 해당 대학생이 관리하는 생명 목록 표시
- 접기/펼치기로 원하는 영역만 집중 가능

---

## 3. 조직 편집 기능

### UI
- 우측 상단에 **[편집]** 버튼
- 편집 모드 진입 시 **드래그 앤 드롭**으로 배치 변경 가능

### 관리자의 편집 권한
- 등록된 대학생을 **자기 밑으로 배치** (대학생 소속 관리자 변경)
- 생명을 **다른 대학생 밑으로 재배치** (생명 담당자 변경)
- 자기 소속 범위 내에서 자유롭게 구조 변경

### 강사의 편집 권한
- **전체 모든 권한** — 어떤 관리자 밑이든 자유롭게 대학생·생명 재배치 가능
- 관리자 간 대학생 이동도 가능

### 드래그 앤 드롭 동작
```
[편집 모드]

대학생 2 를 관리자 A → 관리자 B 밑으로 드래그
→ 대학생 2의 소속 관리자가 B로 변경

생명 김OO 를 대학생 1 → 대학생 2 밑으로 드래그
→ 김OO의 담당 대학생이 2로 변경
```

## 4. 생명 상세 페이지

생명 이름 클릭 시 진입. **대학생과 동일한 구조**:
- 기본 정보 영역
- 일지 목록 (누적)
- 일지 추가 / 수정
- 페일 처리
- 강의 진도표 (강사의 경우 여기서 강의 체크)

관리자/강사는 모든 생명의 상세 페이지에 접근·수정·작성 가능.

---

## 5. 관리자 고유 기능

- 전체 대학생과 생명을 조직도로 파악
- 대학생을 자기 밑으로 배치
- 생명 재배치 (담당 대학생 변경)
- 생명 상세에서 전초 기록 일지 작성
- 단계 변경 가능

## 6. 강사 고유 기능

- **전체 모든 권한** (최상위 권한)
- 조직도에서 모든 생명의 상태를 시각적으로 파악 — 어디서 막히고 있는지 즉시 확인
- **검색 및 필터** 기능으로 특정 생명/대학생/상태를 빠르게 조회
- 생명 상세에서 **강의 체크** 기록 (수강일, 강의자, 비고, 패스)
- 필요 시 즉시 조치 가능 (일지 작성, 단계 변경, 배치 변경 등)

### 강사 검색/필터 기능
- 생명 이름으로 검색
- 상태별 필터 (전초만 보기, 초급만 보기 등)
- 관리자별 / 대학생별 필터
- 오래된 순 정렬 (마지막 만남 날짜 기준 — 관리가 필요한 생명 우선)

---

## 미결 사항
- [ ] 관리자도 검색/필터가 필요한지 (현재는 강사만)
- [ ] 편집 모드에서 실수 방지 (확인 팝업, 되돌리기 등)
- [ ] 대학생이 어느 관리자에도 속하지 않은 경우의 처리 (미배정 상태)
